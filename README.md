## Introduction
`vimhelp-jp.el` is Emacs frontend of [vim-help-jp](http://vim-help-jp.herokuapp.com/)

![vimhelp-jp2](image/vimhelp-jp2.png)


## Requirements

* Emacs 24 or higher
* [request-deferred](https://github.com/tkf/emacs-request)


## Basic Usage

#### `vimhelp-jp`
